# Farkle Scorer

Fast, flexible **scorekeeping for the Farkle dice game** — with tap-to-score dice, manual mode for physical dice, and optional nearby multiplayer.

## Highlights
- **Tap-to-score**: select dice and bank points quickly
- **Manual mode**: enter scores when you’re rolling real dice
- **House rules**: customize scoring and key rule options
- **Optional nearby multiplayer**: connect nearby devices on your local network
- **Offline-first**: no internet required
- **No ads / no accounts**

## Privacy
Farkle Scorer **does not collect personal data**. Game data and preferences are stored on-device.

See [`PRIVACY_POLICY.md`](PRIVACY_POLICY.md).
